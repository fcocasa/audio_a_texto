## Que instalar:

 - pip3 install SpeechRecognition

 - sudo apt install ffmpeg

 - pip3 install pydub

## Ejecutar

Cambiar datos en data.json

python speech.py


### Recomendacion

Crear un ambiente virtual

python3 -m venv /path/to/new/virtual/environment

Sustituir venv por cualquier otro nombre
